## About

A self-study project working through the [Catlike Coding](https://catlikecoding.com/unity/tutorials/) tutorials by Jasper Flick. Each module digs into a different corner of real-time graphics — math visualization, procedural fractals, hash-based noise, and the Unity Universal Render Pipeline (URP).

The point isn't to ship one polished thing — it's to build intuition for *how* shaders, render pipelines, and procedural systems work in a modern engine.

## Modules

- **Graphs** — animated 3D function graphs, exploring how to drive vertex transforms from math.
- **Fractals** — procedurally generated fractal trees, with each iteration spawning rotated children.
- **Hashing & Visualization** — `SmallXXHash`-based deterministic noise, used to visualize hash distributions across surfaces.
- **URP** — building a custom render pipeline pass on top of Unity's URP.
- **Shaders** — handwritten HLSL and ShaderGraph experiments (`Hash Shader.hlsl`, `Hash graph.shadergraph`).

## Stack
